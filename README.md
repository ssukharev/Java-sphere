# Dome Designer (JavaFX)

Desktop-приложение на Java для проектирования металлической полусферы/геодезического купола.

## Технологии

- Java 17+
- JavaFX 3D
- Maven
- Без платных библиотек

## Сборка

```bash
mvn clean package
```

## Запуск

```bash
mvn javafx:run
```

## Интерфейс

- Слева: параметры генерации, кнопки управления, экспорт
- Справа: 3D-модель и таблица стержней (ведомость)

### Редактирование ребер в приложении

После генерации можно редактировать каждый стержень как отдельную сущность:
- смена типа стержня прямо в таблице
- включение/выключение стержня (колонка `Активен`)
- добавление нового стержня по `ID` узлов A/B
- удаление выбранных стержней
- сброс к исходной сетке

Выбранные стержни в таблице подсвечиваются в 3D.
Также можно выбрать стержни двойным кликом ЛКМ прямо в 3D-сцене:
- двойной клик: выбрать один стержень
- `Ctrl/Cmd + двойной клик` или `Shift + двойной клик`: добавить/убрать стержень из множественного выбора

Узлы отображаются только если они принадлежат хотя бы одному активному стержню.

### Параметры

- `Радиус` (должен быть > 0)
- `Количество колец` (должно быть >= 2)
- `Количество сегментов` (должно быть >= 6)
- `Диаметр трубы` (должен быть > 0)
- `Толщина визуального стержня` (должна быть > 0)
- `Высота купола` (`domeHeightFactor`, диапазон `(0; 1]`, по умолчанию `0.5`)
- `Тип сетки`:
  - кольцевая треугольная
  - геодезическая треугольная (упрощенная)

## Управление 3D

- ЛКМ + drag: вращение
- Колесо мыши: zoom
- ПКМ/средняя кнопка + drag: перемещение
- Кнопка `Сбросить камеру`: возврат к исходному виду

## Экспорт

### CSV

Экспортируются:
- узлы (`NodeId,X,Y,Z`)
- стержни (`BarId,NodeA,NodeB,Length,Type`)

### OBJ

Экспортируются:
- вершины (`v`)
- стержни как линии (`l`)
- каждое ребро как отдельный объект (`o bar_0001_horizontal`, `o bar_0002_radial`, ...)
- треугольные грани (`f`)
- именованные группы/объекты для гибкого редактирования в CAD/3D-софте

Файл открывается в Blender.

### STL

Пока реализована заглушка с TODO и понятным сообщением в UI.

## Примечания

- Полюс купола (вершина) создается одной центральной точкой.
- Дубликаты и нулевые стержни исключаются.
- Основание и типы стержней выделяются отдельными цветами.
